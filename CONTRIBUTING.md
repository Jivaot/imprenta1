# Guía de Contribución

## Cómo Contribuir

Apreciamos todas las contribuciones a EstampadosDigitalPro.

### Reportar Bugs

1. Verifica que el bug no ha sido reportado antes
2. Abre un issue con descripción detallada
3. Incluye pasos para reproducir
4. Proporciona ejemplos específicos

### Proponer Mejoras

1. Abre un issue describiendo la mejora
2. Proporciona casos de uso
3. Explica la solución propuesta

### Pull Requests

1. Fork el proyecto
2. Crea una rama (`git checkout -b feature/AmazingFeature`)
3. Commit tus cambios (`git commit -m 'Add some AmazingFeature'`)
4. Push a la rama (`git push origin feature/AmazingFeature`)
5. Abre un Pull Request

### Estándares de Código

- Usa ESLint para JavaScript
- Formatea con Prettier
- Escribe tests para nuevas funciones
- Comenta código complejo

### Guía de Commits

```
type(scope): descripción breve

Descripción más detallada si es necesario.

Fixes #123
```

**Tipos**: feat, fix, docs, style, refactor, perf, test, chore
